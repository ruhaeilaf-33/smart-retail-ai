# 📊 Retail Intelligence System

An **AI-powered Retail Analytics Dashboard** that helps businesses analyze sales data, visualize trends, and make smart decisions using Machine Learning and AI.

---

## 🚀 Features

### 📈 Dashboard Overview
- Total Revenue
- Number of Orders
- Average Sales

### 📊 Interactive Visualizations
- Bar Chart
- Pie Chart
- Line Chart
- Dynamic chart selection

### 🧠 ML Insights
- Identifies top-selling product
- Detects low-performing products
- Provides actionable suggestions

### 🤖 AI Business Assistant
- Chat-based interface
- Answers business queries
- Suggests strategies to improve sales

### 📁 CSV Upload Support
- Upload your own retail dataset
- Real-time analysis

---

## 🛠️ Tech Stack

- **Frontend/UI:** Streamlit  
- **Data Processing:** Pandas  
- **Visualization:** Plotly  
- **Machine Learning:** Scikit-learn  
- **AI Integration:** Groq API (LLaMA 3)

## ⚙️ Installation & Setup

### Clone the repository
git clone https://github.com/ruhaeilaf-33/smart-retail-ai.git
cd smart-retail-ai

python -m venv venv
venv\Scripts\activate

### Install Dependencies
pip install -r requirements.txt
Add API Key

### Create a .env file and add

GROQ_API_KEY=your_api_key_here


###▶️ Run the App
streamlit run app.py
